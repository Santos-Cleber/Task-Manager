# ✅ Task Manager

Aplicação de gerenciamento de tarefas desenvolvida com HTML, CSS e JavaScript puro.

![Screenshot do projeto](assets/screenshot.png)

## 🚀 Funcionalidades

- [x] Adicionar tarefas
- [x] Marcar como concluída
- [x] Deletar tarefas
- [x] Filtrar por status (todas, ativas, concluídas)
- [x] Persistência com localStorage
- [x] Design responsivo para celular

## 🛠️ Tecnologias

- HTML5 semântico
- CSS3 (Flexbox, variáveis CSS, responsivo, BEM)
- JavaScript ES6+ (localStorage, DOM manipulation)

## 💡 Conceitos aplicados

- Metodologia BEM para nomenclatura de classes CSS
- Separação de responsabilidades (storage, tasks, app)
- Conventional Commits
- Fluxo de branches (feature branches + Pull Requests)
- Mobile First

## 📦 Como rodar o projeto

1. Clone o repositório:

```bash
   git clone https://github.com/Santos-Cleber/Task-Manager.git
```

2. Abra o arquivo `index.html` no navegador

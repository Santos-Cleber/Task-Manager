# ✅ Task Manager

Aplicação de gerenciamento de tarefas desenvolvida com HTML, CSS e JavaScript puro.

## 🚀 Funcionalidades

- [x] Adicionar tarefas
- [x] Marcar como concluída
- [x] Deletar tarefas
- [x] Filtrar por status
- [x] Persistência com localStorage

## 🛠️ Tecnologias

- HTML5 semântico
- CSS3 (Flexbox, variáveis CSS, responsivo)
- JavaScript ES6+

## 📦 Como rodar o projeto

1. Clone o repositório:

```bash
   git clone https://github.com/seuusuario/task-manager.git
```

2. Abra o arquivo `index.html` no navegador
